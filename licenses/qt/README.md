# Qt 6 ライセンス告知

YAPCRPlayer は Qt 6（Core / Gui / Widgets / Network / Core5Compat、および
`windeployqt` が収集するプラグイン）を **動的リンク**で利用し、配布物に同梱しています。

- 利用ライセンス: **GPLv3**（Qt のオープンソース提供のうち GPLv3 オプションを選択）。
  YAPCRPlayer 本体の GPL-3.0-or-later と同一系列であり、結合に制約はありません。
- 使用バージョン: **Qt 6.11.1 `msvc2022_64`**（`docs/build.md` 参照）。
- ライセンス全文: 本ディレクトリの [`Qt-LICENSE.txt`](Qt-LICENSE.txt)
  （Qt インストールに同梱される `Licenses/LICENSE`。GPLv3 / LGPLv3 の全文を含む）。

## 対応ソース（GPLv3 §6）

Qt の対応ソースは The Qt Company / KDE が公開する公式ソースアーカイブから入手できます。

- ソースアーカイブ: https://download.qt.io/archive/qt/6.11/6.11.1/single/
- 詳細・書面オファーは `CORRESPONDING-SOURCE.md` を参照。
